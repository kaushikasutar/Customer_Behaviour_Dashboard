# Customer Behavior Dashboard

## Overview
This project analyzes customer purchasing behavior using Python, SQL, and Power BI. The objective is to uncover insights related to customer demographics, purchasing patterns, product preferences, discounts, shipping methods, and review ratings.

The project follows a complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard creation, and business reporting.

---

## Dataset

The dataset contains customer transaction information, including:

- Customer ID
- Gender
- Age
- Item Purchased
- Category
- Purchase Amount
- Review Rating
- Discount Applied
- Shipping Type
- Payment Method
- Subscription Status
- Frequency of Purchases

## Project Workflow

### 1. Data Loading

## How to Run

### Python Analysis

1. Clone the repository

```bash
git clone https://github.com/yourusername/Customer_Behaviour_Dashboard.git
```

2. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

3. Run Jupyter Notebook

```bash
jupyter notebook
```

### SQL Analysis

1. Import dataset into MySQL.
2. Execute queries from:

```sql
customer_queries.sql
```

### Power BI Dashboard

1. Open the `.pbix` file.
2. Refresh data source if required.
